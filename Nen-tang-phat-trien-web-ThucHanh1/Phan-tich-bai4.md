Phân tích:

Form gồm 4 phần chính, được ngăn cách bằng dòng kẻ:

1: Basic info

+id: text read only

+last/ first name: text

+gender: radio

+title, suffix: text

+birth/hire date: datetime-local  
+SNN: text  
+report: select

2:contact info

+email: email

+addres/city/region/postal code: text

+country: select

+home phone: tel

+photo: file

3: optional info

+notes: textarea

+preferred/active: checkbox

+capcha text box: text

4:footer

+submit: submit

+cancel: button

+mũi tên trái/phải: button
